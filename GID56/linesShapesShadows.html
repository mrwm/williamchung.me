CR-10

I got myself a new 3D printer! As the title suggests, the new printer that I got is the CR-10. I got this printer as an upgrade from my previous printer, which was the Tronxy XY100. The CR-10 is much larger, but comes with the downside of being louder. In order to reduce the sound, I plan to install dampeners on the motors in order to reduce the vibrations. For the specs of this printer, it has a bed size of 300mm (X) × 300mm (Y) × 400mm (Z), a heated bed, and a spool holder attachable to the control box.
   So far, the only things that I have printed are the upgraded feet for the control box [upgrade found here] and the demo file that came preloaded in the included mini SD card. Besides using the CR-10 to print upgrades for itself, I also took apart the control box and flashed the unified firmware that can be found here. I plan to print some more upgrades to the printer before I start printing other things with this.
